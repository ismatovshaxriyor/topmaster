# Catalog bo'limi

Bazaviy yo'l: `/api/v1/catalog/`

Ushbu bo'lim platformaning ommaviy ma'lumotnomalari — shaharlar va xizmat
kategoriyalari — uchun o'qish uchun-only endpointlarini o'z ichiga oladi.
Barcha endpointlar ochiq (autentifikatsiya talab etilmaydi).

## Endpointlar

| Metod | Yo'l          | Fayl                           | Tavsif                              |
| ----- | ------------- | ------------------------------ | ----------------------------------- |
| `GET` | `cities/`     | [cities.md](cities.md)         | Barcha shaharlar ro'yxati           |
| `GET` | `categories/` | [categories.md](categories.md) | Faol xizmat kategoriyalari ro'yxati |
